# NMSU Senior Project Spring 2018

### Authors: 
Cyrille Gindreau  


### Dependencies
NodeJS 8.9.4  
https://nodejs.org/en/


### Project setup
Clone repo: `git clone https://github.com/cyrillegin/seniorProject.git`  
Change into directory: `cd seniorProject`  
Run setup command: `npm run setup`  
Start dev server: `npm run start`
